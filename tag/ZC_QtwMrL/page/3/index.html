刘润真正的利他

2019-08-06 #刘润
刘润 : 《刘润：真正的利他，是一种在细微处都能开花的修养》一个人的境界，可以从他的言语中感受出来。提问：你一位朋友，在微信上激动地对你说，&quot;你知道吗，美元对人民币已经破7了&quot;。而这则消息，你昨天就知道了。你会怎么回答？下面的三种回答，代表了三种境界。1）我早知道了。这么回答时，你内心的BGM大概是这样的：朋友圈昨天都已经刷完屏了，你今天才知道啊？这也太晚了吧！这时，如果你愿意，你可以跳出自己，在半空中旁观自己。你会看到一个略带优越感的自己。你会忍不住脱口而出：我早知道了。这句&quot;我早知道了&quot;，背后是一种你可能藏在底层潜意识里，你的上层意识没有识别到的情绪：怎么样？我比你消息灵通吧？我比你优秀吧？这句话一说出口，你的小小的&quot;自我&quot;（ego），就闪了一下光，轻轻地刺了一下你朋友的眼睛，刺了一下他的好意。2）确实如此。这句话的境界，就比“我早知道了”高一个档次。为什么？因为如果你站在半空，你会看到顺着这句话飘到你朋友那里的信息流，不是“我比你聪明”，而是“你是对的”。不要小看这句话。能说出这句话，说明你已经挣脱了小小的ego的束缚，能够把心思放在别人身上。所谓的教养，就是让别人觉得舒服。如果你被ego束缚，强行让别人舒服的代价，就是自己不舒服。你会觉得：他不知道我比他早知道，这太亏了。我要让他知道。算了算了。为了他舒服，我忍住不说吧。这不是教养。这是假装的教养。你让他舒服后，你也很开心，这才是教养。能做到这点的前提是，你已经挣脱了自己的ego。这就是为什么我说，能发自内心说出这句话，段位高一个档次。但，这不是最高级的。3）收到，谢谢。最高级的表达，是说：收到，谢谢。为什么？因为这时，你关心的已经不是他说的话本身，而是他说这话背后的动机，和情绪。他为什么要告诉你“汇率破7”了？因为他关心你。他觉得你有必要知道。他怕你不知道。你如果把心思放在他的动机和情绪上，就会满怀感激。这时，你是不是更早知道，还重要吗？一点都不重要。表现自己聪明重要嘛？一点都不重要。还很傻。你应该由衷地谢谢他。你应该鼓励他。这样他才会在下一次遇到类似情况，会想起来，你应该知道，会想起来告诉你，而不用担心：会不会他已经知道了？所以你怎么回答？我知道了，但还是谢谢你。这么表达进步很大，但还是会让他有些沮丧。你要说：收到，谢谢。你不用说：要不是你告诉我，我还不知道呢。你不用这么说。这么说是撒谎。但你可以忽略你已经知道这个情节。因为它不重要。“收到，谢谢”是情绪，是鼓励，是事实。最后的话：今天为什么要写这篇文字？很多人说，“利他，是最伟大的商业模式”。但我想说，真正的利他，不仅仅是我带你分钱。真正的利他，源自一种根植心中的价值观，一种在细微处都能开花的修养。你的员工，你的合作伙伴，你的客户，不知道为什么，就是很喜欢和你在一起。你也不知道，为什么你就有一种吸引力。这种吸引力不是装出来的，而是突破了自己那点ego之后的修养。2019-08-06点赞数:133评论区：深爱巴蒂 : 档次不一样 Answer : 收到，谢谢。 旅行君主✅ : 好的 再见陛下 : 我会说：啊是嘛(≧∇≦*)，我刚也听说了，咋回事呀。这是啥心理呀 楼楼楼 : 学习了 张勇|健康营养|杭州 : 确实，很多时候跟事实真相相比，我们所说的话会对方情感和心理带来什么样的影响更重要 武恒标 : 收到，谢谢 工 : 狠斗EGO一闪念到这个地步，我的天! 谢谢润总，我知道了[微笑]

昨天我们在一个群

2019-08-06 #刘润
刘润 : 昨天，我们在一个群里讨论老人带孩子的问题。有群友认为，之所以不让老人带孩子，是因为我们父母这一代人，其实没有真的&quot;带&quot;过孩子，没有经验。老了后，不可能立刻成为专家。有群友认为，父母，和爷奶关于带孩子的争夺，其实是权力之争。谁带孩子，谁就掌握了下一代。比如过去某个朝代，皇妃生子要杀掉皇妃。因为怕她们权力过大。 这些观点，我觉得都挺有道理的。但我认为，我们和爷爷奶奶关于教育理念的不同，也许还有别的原因。我们30岁生孩子，父母60岁。一个是而立之年，一个是耳顺之年。而立之年的你，最看中的东西是什么?成就，责任。你如果不思进取，就被陪朋友们耻笑，同事们排挤。你知道竞争的惨烈，你知道成就的重要。同时，成为父母的那一天，你正式长大了。你更有责任感，你知道肩负重任。你知道成为一个具有完全人格的人，有多重要。你要你的孩子，也有责任感。所以，你所有的教育导向，都指向于此。你以为，这就是正确的。但是，耳顺之年的父母呢。他们最看重的东西，是健康，是快乐。 你所经历的一生，他们都经历过。你认为的轰轰烈烈，他们都体会过。他们也曾认为成就和责任高于一切。可是，老了之后，他们开始意识到，那一切曾以为高于生命的意义，其实没有生命重要。他们开始顿悟：平平淡淡才是真。 所以，他们的教育导向，就变成了吃饱了，长高点，多睡会儿，别被人欺负，比什么都重要。你说，你们这是不对的。你们不会带孩子。 但是，你想过没有，你，不就是他们带出来的吗？你的优秀程度，就是他们带孩子的水平的优秀程度。他们不是不会带孩子，只是他们的价值观发生了变化。某某的父亲是军人，小时候对他的教育非常严厉。今天他非常感谢父亲。但是现在，父亲对孙子却百般溺爱。你都看不懂。为什么？是他突然忘了怎么带孩子了吗？并不是。而是因为他开始认为那些事情不重要了。健康，快乐才重要。 同样道理。觉得这重要、那重要的你，今天对你孩子照英文书养。但是等你老了，面对孙子时，那种一个走向尽头的生命，看着一个新生的生命的喜悦，会让你忘了书里所有的原则。所以，关于带孩子，不是老人&quot;会不会&quot;的能力问题，而是老人&quot;愿不愿&quot;的心态问题。教育，基本都是投射。你希望把而立的自己投射在孩子身上，还是把耳顺的自己投射在孩子身上？ 最后说一句。我们都会变。我们老了会觉得自己年轻时傻得可笑。但其实一点都不可笑。你再年轻一遍，还是会那样。 年轻的你呢，也会觉得你父母蠢得可爱。但其实一点都不会蠢。你老了，也会那样。这不是进步。这是轮回。周而复始，生生不息。2019-08-06点赞数:121评论区：谢鹏 : 的确如此，我自己对父母态度的转变，正是从自己有了孩子以后开始的！不养儿不知父母恩，说的太对、太好了！[拥抱] 小小朱 : 确实如此，人在不同阶段，认识与行为就不同。就如人在爬山，爬到不同位置，看到的风景也是不一样的 嘉玲 : 投射 这个词用得太精准了 jobs潘｜儿童保 : 分析的好，但是我还看重一点，很多时候老年人带孩子，不讲就卫生，我父母还好，但是很多孩子家里老人又要带孩子又不想太累，经常带孩子去各种集体儿童游乐设施玩耍，要知道各种病毒细菌在那里互相交叉感染是最厉害的，流感爆发也是最容易受感染的区域。 所以，我的孩子我可以控制的很好从来不去那些地方玩，但是别人家呢？他们去玩生了病上幼儿园也会传染到我的孩子。所以，过去两年，我一直在班级群里充当卫生监督的角色，在群里经常发儿童医院人满为患的视频，给这些家长看，然后我说：你们看看哦~~，想去排队看病的，就让老人多带孩子去游乐场所多玩玩😂 没想到效果还真不错，过去的一年时间班级里很少有小盆友流感了😂😂😂😂 郭惠 : 我的孩子一直母亲带，也有时思考过这个问题，但后来一想，母亲带比我们有经验多了，毕竟她也带出了如此优秀的我们[偷笑][偷笑]，只是有一点，自己也要多和孩子交流，多创照一些让孩子了解这个司机机会 Lucky琳|星空画* : 对啊 谁都会老 不要嫌弃自己的父母唠叨 那是他们爱的表达。或许等我老了 我也会唠叨 像爸妈一样 哈哈哈 多一些理解和包容就好了 加乘 : 这个还真不好说，毕竟30年前的那个时代和现在的时代不一样，工作和生活方式也不一样，承受的压力也不一样。等我60的时候带孩子还真不确定跟现在我的父母是不是一样。哈 荧光 : 真的是这样，每个年龄阶层看重的东西不一样，导致教育方式的差别

2019-08-04 #刘润
居尔提 ma0x : 最近回去翻了几本看过很多次的书，赚钱的两种策略，赚快钱与挖金矿。经济学基本原理是，任何一个行业，长期来看是没有超额利润（easy money)的。你的销售额只能覆盖你的成本，包括原材料成本，销售成本，人力成本等等。如果有经济利润，其他人就会加入这个行业，供给增多，价格下降，直到没有超额利润。也就是说，如果一门生意好赚，其他人就会一直涌进来，直到不好赚为止。除非你能用“其他方式”锁住门槛，变成特许行业，不然什么行业最终都会没有赚头。依照这个原理，赚钱只有两种方向：1.机会财：不停的寻找新的赚钱机会，在其他人来到之前卡住位置。等其他人涌入，就把锅甩给他们，自己换一个战场。我把这个在这叫“赚快钱”，短期收入。2.经营财：能把成本降的比同行更低，在同一个领域中不断的踩死竞争者。我把这个在这叫做挖金矿，长期收入。照这个原理来分析，多数人的创业，都是想赚快钱...做网拍也好，眼镜店也好，奶茶店也好，在夜市摆摊卖消夜也好，或者骑车送餐也好，其实都是靠速度赚钱。因为，这些是没有门槛的行业，长期来看净收入不可能太高。这种类型的工作，在初期的机会财阶段结束后，会有大量的同行出现。想赚后期的经营财，在不违法的前提下，能赚的都是辛苦的皮肉钱。那该怎么办？我们把赚钱方式分成短期与长期来分析：先说赚快钱，短期收入：短期收入跟长期收入不同！一是可能存在竞争不完全的市场，这个时候新市场的出现带来的需求大于供给，只要你做，就能赚钱。因为市场做出调整需要时间比如，你比外卖大军更早开始送外卖，由于当时人没那么多，你还真的赚进月薪两三万。理由并不是你有多厉害，纯粹是因为竞争很少，好赚的钱没人抢。二是存在资讯不对称，人们没有发现你做的事情能赚钱。这个必须有背景，有条件，没有的人不要妄想。比如你家里有一个特殊配方，或者你搞出一个新东西。别人要想也赚一手，必须花很多时间研究。你做这个，一定时间里面可以类似于垄断的赚钱。所以，寻求短期收入的思路有二：第一，寻找一个新兴行业，投入进去做。第二，如果自己有什么别人没有的资源，想办法用来赚钱。但很容易可以看出来，这两点都不简单。投入新兴行业需要钱和一定的资源，发现新市场也需要成本。至于自己想搞出别人没有的东西，更是困难。而且，短期市场最大的问题是，不确定未来会怎么样，从某种程度上说，投入短期市场的人，最后更多积累的是做生意的经验，想长久做下去，没有保障。再来是挖金矿：长期来看，想赚钱必须要提高个人的价值。个人的价值不一定是说技能高低，比如你学某种技术做到行业第一，这是一个方向。你掌握的资源量也是你的价值，比如你要做手机，你得首先能找得到有关的供应商，设计师，程式设计师。如果你能找得到，相比于那些找不到的人，你的价值就高多了，这其实就是人脉、通路。总结一下，如果你现在一无所有，想要赚大钱，长期来看没有捷径，两个思路。1.好好读书，学习真正有用的技能，吃技术饭，多年钻研成宗师。2.累积资源和人脉，进一个行业把各相关的产业链都做熟做好，相对于那些没有这些经验的人，这就是你的收入。千万不要想著有什么捷径，更不要唱高调。如果你家里有钱有资源有人脉，这就是你的捷径，果断运用。如果你家里没有什么可以支持的，多读书，然后找一个正职工作好好干，累积资源再创业，可能最有效。#end按赞打赏是持续输出价值的动力，写这样一篇帖要花一定的时间精力，我在这发的帖都是首发或只发在这，大家去看各个星球的状况，我这样做显然不太一样，单纯分享有价值内容。2019-08-04点赞数:129评论区：K : 请问一下，那几本看过很多次的书是什么书？谢谢。 刘润 : 写得非常好。用熊彼特的话说，“只有创新才能带来利润”。我把它分为：红利，工资，利润。以后写文章与大家分享。打赏了。 居尔提 ma0x : 谢谢刘老师 刘润 : 给你私信了。求加微信。 居尔提 ma0x : 收到，谢谢老师 居尔提 ma0x : 1. 食利者政治经济学（中文译名） 2. 利润率的政治经济学 小张 : 有点像 受阻模型组 飞越疯人院 : 要不自己是金子，要么就是能发现金子
