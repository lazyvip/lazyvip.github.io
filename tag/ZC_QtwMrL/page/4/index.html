我加入微软时曾被

2019-08-02 #刘润
刘润 : 我加入微软时，曾被一句话深深激励：我们的产品，改变着上亿人每天的生活。是的。这句话反过来讲就是：上亿人的生活，就在产品经理们的一念之间。给你讲个小故事。今天发生的故事。我今天出差，从上海去杭州。因为一些原因，到高铁站早了。于是想，能不能改早一班，提前到杭州，入住酒店。时间花在喧闹的高铁站，不如花在安静的酒店。还能干点活。于是我去排队。虹桥站内有“售票”，“改签”和“退票”三种窗口。我拿着票，排进了“改签”的队伍。终于排到我了。我说请帮我改最早的去杭州的高铁。改签员看了一下屏幕，说：你的票已经改过一次了。不能再改了。是的。我的行程一直变化中，助理已经帮我改过一次行程了。我说：那怎么办？她说：只能退了再买。我说那就帮我退了再买吧。退票费我付给你。改签员说：我们只改签，不退票。退票在另一个队。好的。一年出差100多天，对这样的“游戏设定”，我已经被磨练的毫无怨言。没搞清楚状况，是我的错。我的错，我的错。我赶快排入了“退票”的队伍。终于排到我了。我说请帮我先退票，再买最早去杭州的高铁。退票员说：我们只退票，不卖票。你退不退？我说：啊？那你先帮我看看，前面班次还有座位吗？有座位我就退。免得退了才发现没票。退票员说：我们看不到。只负责退票。你要查有没有票，要去售票窗口查。我问：那我查完能直接买吗？退票员说：不能。要退掉才能买。有票在手，是不能再买附近时间的票的。我的头“嗡”地一声。这不就死循环了嘛？我去排“售票”的队，查到前面班次有票，但不能买。回来退票。退完后再排队去买票。那时候没票了怎么办？没办法。只能这样。这是谁设计的？产品经理。我能理解，产品经理的每一个设定，都是有道理的。比如，有票不能再买，是为了防止一个人买好多票，真正想走的人没票。只能改签一次，是为了车票安排的稳定性，最优化全网行程。等等。但是，这些放在一起的复杂性，让产品经理忽视了“二次改签”的悖论。产品经理办公室一念之间的疏忽，让我在高铁站站在三个窗口前进退两难。但是，活人怎么能被尿憋死。我突然想起来，自动取票机上，也能查询余票啊，还不用排队。我立刻冲到自动取票机前。刷身份证一查，果然前面一班还有两张商务座。从自动售票机转身回去退票，再回到自动售票机上买票，大约要5分钟。这时，我要做个决定，要不要冒一个险：退完票的5分钟内，这两张商务座正好都被卖完。应该不会吧？要卖完早卖完了吧？不会等到这5分钟吧？我决定冒这个险。我冲回最短的退票窗口，退了票。退票员收了我20%退票费。她还要撕发票给我。我不要发票，只想她把身份证尽快还给我。拿到身份证后，我立刻冲到自动取票机前，非常熟练地开始买票 …… 你猜结果怎样？就在这5分钟内，两张商务座都！卖！完！了！天啊，运气太差了吧！我压制住自己的惊讶，提醒自己意识到，我正处于完全没票的状态。于是，我赶紧搜索后面的班次，发现后面班次全部没票，除了突然出现的唯一的一张商务座。我用迅雷不及掩耳盗铃之势，买下了这张票。长舒了一口气。差点去不了杭州。这时才看一眼车票，是哪一班？什么时间的？天啊 …… 这张票，居然就是我刚！刚！退！的！那！张！票！我排了3次队，终于买回了自己进站时就在手上的票。什么都没发生，除了我少了40元钱。我突然觉得，这是一次有趣的人生经历。我决定拍张照纪念一下。回到最开始。为什么我说，上亿人的生活，就在产品经理们的一念之间？因为产品经理们，只要一念之间，就能节省我今天的30分钟和40元钱。小张心情低落地坐在电脑前。他和女朋友吵架几天了。怎么道歉都不管用。他倒了咖啡，一口没喝。心想，几年的感情，是不是就这样分手了。突然电脑上，女友发来了消息：我原谅你了。小张欣喜若狂。太好了。他心底暗暗发誓，以后一定要珍惜。突然，突然，一念之间，他脑海里蹦出一个公式：“二次改签 我试一下你家婚纱

2019-08-02 #刘润
想YAO说说看的姚祥 : 我试一下你家婚纱，你却要收我1000块试穿费？（关于价格的一些事）看到这个标题，我相信很多办过婚礼的人都感同身受。新人满心欢喜的去试婚纱，普通婚纱头3件试穿免费，之后的500元可以再试3件，高定的话则需要1000元试穿1件。虽然婚纱店的销售会说：“如果您确定了用咱们家的婚纱，1000块钱可以抵扣在总额里的。”但是你还是会感觉不舒服。我来看你家婚纱是看得起你，凭什么试婚纱还要我花钱？而换一个角度看，我觉得这个试婚纱需要收费的行为是很合理的。因为商家用这种试婚纱需要付费的门槛有效的区分了真正的消费者和只是试着玩的路人，只有对这婚纱感兴趣的人才会继续和销售聊下去。这就是“价格歧视”，用价格区分真心的消费者和非消费者。“价格歧视(price discrimination)”中的“歧视(discrimination)”是一个中性词，并没有贬低的意味，而是应该被理解为区分。如果你觉得婚纱店只是用这一千块钱试探你是不是真的消费者，那就大错特错了。他们同时也是在试探你的价格预期和心理承受能力。因为土豪和普通人听见“1000块试穿一件婚纱”时的反应肯定是不一样的。普通人多半会沉默，然后盘算划算不划算。而土豪则有可能会想无所谓，试一下也无妨。我这说的土豪不是真的挥金如土的人，而是觉得1000块试穿一件婚纱是可以接受的人，因为他们的心理预期可能是1500元试一件婚纱。通过上述的人设分析，销售人员就可以对症下药了。在服务后的讨价还价阶段，销售就可以“抬高价格”让愿意花钱的客人更多花钱，也可以增加“折扣力度”让不愿意多花钱的客人少花钱，毕竟他们的目的是挣到钱。综上所述价格是存在歧视性的，一个价格可以区分真正有意愿的消费者和吃瓜群众。所以，这篇文章希望可以给大家另一个角度来审视消费过程中“无故的高价”，看清套路。2019-08-02点赞数:24评论区：想YAO说说看的姚祥 : 欢迎大家关注我的微信公众号：想YAO说说看 也希望大家对我的文章多评论，互相交流学习🙏🙏🙏

这个时代赋予了中

2019-08-01 #刘润
刘润 : 这个时代，赋予了中国3次人口红利。每次人口红利，都洗牌了中国商业世界，成就了新的商业巨头。这3次红利，你抓住了哪一次？第1次，叫“供给侧人口红利”。根据毛大庆的数据，在中国执行计划生育国策之前，1966-1973，这8年间，中国出生了3.1亿的人口，占今天中国总人口13.6亿的22%。3.1亿是什么概念？这个数字超过绝大多数国家的总人口。到了80年代-90年代，这批人口大约20多岁。充沛的劳动力供给，导致人工价格极其便宜，中国在劳动力密集型的制造业，产生了难以复制的比较优势。毛泽东时代的英雄母亲们“生育了”中国劳动力，而这些劳动力“造就了”世界工厂。海尔，联想，都是这个时代的代表。劳动力难以复制的比较优势，导致中国的经济突飞猛进地发展。改革开放40年，中国年均GDP增速8.5%。其中相当长一段时间，年均GDP增速超过10%。第2次，叫“需求侧人口红利”。虽然1975年左右，中国开始实行计划生育政策，出生人口开始下降，但因为出生率高于死亡率，所以总人口还在上升。今天中国的总人口，已经将近14亿。而1949年建国时，中国人口只有5亿多。这巨大的需求侧人口增加，极大刺激了90年代-00年代的一个新兴行业的发展：互联网。互联网的发展，严重依赖基于人口规模的“网络效应”。只有8000万人口的德国，互联网很不发达；只有1.2亿人口的日本，互联网不算发达；拥有3.2亿人口的美国，互联网非常发达。而中国呢？我们有14亿人口。根据腾讯官方数据，微信现在的月活（MAU，每月至少使用一次微信的用户），已经超过10.98亿。 10.98亿！这个数字，比美国、日本、德国的人口总和，还要多1倍。互联网的发展，和科技精英们的努力分不开，但首要还是得益于中国供给侧的人口红利。阿里，腾讯，都是这个时代的代表。2018年，全球市值最高的10家公司中，阿里、腾讯，拿走两席。第3次，叫“高素质人口红利”。中国大学每年都在扩招。从1978年的千军万马过独木桥，到2018年的条条大路通罗马，40年来，中国共有1亿人左右，接受了大学以上的高等教育。预计再有40年，这个数字会变为2-3亿。上大学有用吗？这是过去10年，大家讨论非常多的话题。大学生越来越多，越来越不值钱，毕业了还不如快递赚得多。当低生育率导致“供给侧人口红利”消失，快速老龄化导致“需求侧人口红利”减弱，我们突然意识到，这1亿大学生的价值。未来中国的竞争力，不是低成本制造业，不是大规模互联网。未来中国的竞争力，只有可能是真正的技术创新。谁能做出最智慧的人工智能？最精密的机器人？最有价值的基因技术？最多的通信专利？最干净便宜的能源？谁能做出来，谁就拥有支配世界的权力。而这一切，都靠受过高等教育的国民。华为，是这个时代的代表。但是，这时代不能只有一家华为。时代需要，并会奖赏更多的抓住“高素质人口红利”的企业。时代需要，并会奖赏更多的华为。被第3次人口红利奖赏的，下一个华为，会是你吗？2019-08-01点赞数:111评论区：非洲鼓中国周周 : 学习 知识 技术的重要性 张典 您的私家网络顾* : 下一个时代5G技术非常看好 贾颜涛 专注积分落地 : 是的。 北方 : 虽然高中毕业，虽然已经失去了进去华为这样公司的机会，但命运掌握在自己手上 张志芳 : 中国平安会算一个吗？2018年全球金融科技发明专利中国平安排第一，感觉都快成一家科技型公司了。 猫掌门~进化记 : 红利，短暂的供需失衡！对不对？ 努力中的人 : 大家觉得下一个是谁 楼楼楼 : 二八定律百分之八十的大学也会非常平庸

昨天发了一篇文章

2019-07-30 #刘润
刘润 : 昨天发了一篇文章，讲到阿德勒的“课题分离”理论，说判断一件事是谁的错的标准，是：谁的损失大，就是谁的错。什么意思？比如，有人在地铁里踩了我一脚。谁的错？我的错。啊？明明是他踩了我，为什么是我的错呢？难道我不应该要求他道歉吗？你可以要求他道歉。但是，道歉有什么用？而且，你要求他道歉，不需要花时间嘛？他耍无赖和你吵起来，不更需要花时间嘛？你的时间没地方花了吗？这时，对方可能反咬一口：你怎么把脚乱放啊？！你怎么办？你要说：我的错，我的错。然后，心平气和地走到旁边。因为，你的时间，比他值钱。这就是“谁的损失大，就是谁的错”。但可能还是有同学不完全明白。今天多讲两句。一个人心中，应该有三种“对错观”：1）法学家的对错观；2）经济学家的对错观；3）商人的对错观。什么意思？我一个个来讲。1）法学家的对错观。我举个例子。一个坏人A，诱骗好人B到没有锁门的工地C，然后B失足摔死了。请问，这是谁的错？这当然是A的错，这就是蓄意谋杀。这还有什么好讨论的。是的。如果证据确凿，从法学家眼中看，这就是A的错。但是，这种大快人心的对错观，不一定能防止类似案件下次发生。法学家做不到的事情，经济学家也许能做到。2）经济学家的对错观。经济学家有不同看法。B被坏人A，诱骗到没有锁门的工地C摔死了，是C的错。啊？为什么啊？这也太冤了吧？这是因为，整个社会，为了避免B被A诱骗到工地的成本，比工地C把门锁上的成本，要高得多。如果惩罚了工地，虽然工地觉得冤，但是以后工地就都会把门锁上了。这样的事情，会大量减少。所以，经济学家，是从“社会总成本”的角度，来判断一件事的对错在谁。虽然听上去不合理，但有时比纯粹的“道义”更有效果。3）商人的对错观。但不管是A的错，还是C的错，B都死了。把谁抓起来，B都无法起死回生。所以，从个体利益最大化的角度，B只能怪自己。我的错，我不该蠢到被A欺骗。只有这样的认知，才能保护B自己。一个人走在人行横道上，一辆卡车呼啸而来。所有人都大声呼喊他让开。他淡定地说：他不能撞我。他撞我是违反交规的，他全责。我就不让。最后这个行人被撞死了。这是谁的错，卡车司机的错？当然。但是这样的认知无法救命。这个行人要认识到：这是我的错。我应该让开。因为死的是我。你看法官认为A错，经济学家认为C错，商人认为B错。这就是一个人心中的三种对错观。如果你是评论家，你可以选择法官的立场；如果你是政策制定者，你应该选择经济学家的立场；但如果将要失足摔死的就是你自己，我建议你选择商人的立场。我的错，我的错，都是我的错。因为我损失最大。谁损失大，就是谁的错。2019-07-30点赞数:119评论区：福州陈巍 : 不同角度切入点不一样 旅行君主✅ : 明白了，有道理 海东 : 人生观，价值观，世界观，还有对错观，消费观，好多观啊 李崧 Stanley : 火车疾驰而过引起的火星燃烧了铁轨旁边晾晒的棉花，谁的错？经济学家说是棉农的错，因为他避免损失发生而采取的成本，比整个社会为避免之而采取的总成本，要少得多。从商人的角度，还是棉农的错，可预见后果发生时谁的损失大就该怪谁。 有脾气 : 这观点够味儿，确实如此，利益相关度决定立场，每个人描述的都是自己眼中的世界。 焦浩 : 昨天看《哈佛商业评论》里有一期专门讲时间的文章《幸福的时间》写的很不错。欢迎大家查阅。 工 : 不服不行，一个小题目，挤压出一个大道理。学到一点思考方法：一个问题如果想不清楚，一定是因为问题本身分的不够细。 自强不息 : 副驾驶的乘客，总是说“干嘛让别人插车，撞上去，他全责，他全责” 我淡定刹车：他全责又怎样，陪他报保险的是我，去修车的是我，找他理论赔偿的还是我，是我，是我，还是我。

咨询顾问的主要工作

2019-07-29 #刘润
刘润 : 咨询顾问的主要工作之一，是给企业提建议。建议，必须针对问题；问题不同，建议也不同。我把建议，大体分为三种：救命建议，治病建议，和养生建议。1）救命建议。突然，一位同事晕倒在你身边？你应该怎么办？立刻拨打120，送到最近医院。这时候，你会说等等，我查一下，哪家医院处理这种突然晕倒类问题最擅长吗？你会看看最牛的主任医师哪天上班吗？你不会。因为，这时候“治病”不是最重要的，你第一要务是救命。同样道理，一家公司现金流断裂，你给他的建议应该是“提炼愿景、使命、价值观”吗？不。这时，你应该卖掉或者关闭不赚钱的业务，甚至裁员，阻止大动脉失血。可是，关闭业务、解雇裁员，会有副作用吧？IBM传奇总裁郭士纳接掌帅印后，裁掉几万名员工。被裁掉的人，一定是怨声载道；留下的人，一定是士气大落。但是，连命都快没有了，救命第一。救命，大于治病。2）治病建议。到了医院，命保住了。这时，你问医生怎么回事。医生说，他晕倒是因为一种罕见心脏病。然后，你开始和医生讨论这种病的治疗方案。是搭桥，是移植，还是装人工心脏。如果是装人工心脏，哪家医院手术的成功率最高。这时候，你会对这个病人说“少油少盐多健身，就没事了”吗？你不会。因为养生习惯、保健食品，也许可以防病，但是不能治病。病一旦发生，只能积极医治。同样道理，公司产品质量出现严重问题，导致大量客户投诉、退货。你给他的建议应该是“全体员工加强质量第一的文化建设”吗？不。这时，你应该召回产品，弥补客户损失。可是，召回产品，也是有成本的吧？当然。但如果不这么做，公司声誉将会大损，病会越来越严重，最后要你的命。临时抱佛脚讲文化，治不了病。治病，大于养生。3）养生建议。你的朋友出院了，问医生有什么建议。医生说，饭后百步走，能活九十九。合理饮食、无烟无酒、心情愉快、适当运动。这些都是养生建议。没有命悬一线、病入膏肓的人，感受不到这些建议的价值。同样道理，公司在“健康”时，我们要关注客户，投入研发，公平激励，建设文化。这些听上去很虚，什么效果都没有。但却可以治未病。一个优秀的商业顾问，会冷静清晰地知道，什么时候给什么建议。救命时，冷静地壮士断腕；治病时，咬着牙刮骨疗伤；养生时，坚持要跑步爬山。而一个糟糕的商业顾问，在顾客命悬一线时，建议你治病；在病入膏肓时，劝解你养生；在一切正常时，建议你饮鸩止渴。而你，必须具备清醒地知道自己处于哪个状态的能力。2019-07-29点赞数:95评论区：slj : 药💊，工具，解决方案都在库里，但何时用，量多少，怎么用，才是最难的抉择的。想想都头疼！ 邱斌 : 怪不得都要看情况呐 方一茗 : 救命是没做重要紧急的事 治病是没做重要不紧急的事 养生是做了上面两个事，但做的不透不持续 刘萍 : 浅显易懂，大道至简。 Vivian : 含金量太高，我要好好地琢磨消化。 腾飞盛业-ALEX : 养生之道何其多，找到正确的，坚持不懈做，很难[呲牙] 王天海 : 发现问题，提出问题，分析问题，解决问题。以问题为导向，以问题为核心 Teh 郑 : 简单易懂，背后功夫少不了

什么是经验经验重

2019-07-29 #刘润
刘润 : 什么是经验？经验重不重要？经验的本质，是一套“历史反馈系统”。什么叫“历史反馈系统”？人们只能从反馈中学习。以前做过，所以怎么做能成，怎么做会失败，就会用“因果链”的方式，存在大脑中。虽然这些“因果链”不一定完全正确，但是却多少有些价值。然后，今天你想做一件事，刚做了一半，那个存储的历史中的因果链就跳出来了，用“经验”告诉你，这个“因”，通常会得到什么“果”。这就是“历史反馈系统”。因为是亲身经历，所以印象深刻，但它的问题时，经验未经提纯，“因果正确性”非常不稳定。可能有用，可能没用，甚至可能会害了你。那学习呢？学习的本质，是一套“他人反馈系统”。向其他创业者学习，相当于“复制/粘贴”他们的“历史反馈系统”，然后存在自己的记忆里。这就节省了你自己亲身经历的成本，但也有两个问题：1）印象不如自己经历深刻，2）他人的“历史反馈系统”，“因果正确性”也非常不稳定。所以，你还可以向智者学习。这些智者，有可能亲身经历，也有可能仅仅是从别人的经历中提炼。但是，他们有一种能力，就是把他人，或者自己的“历史反馈系统”提纯、强化，从而得到非常强壮的“因果链”。这些被提纯的、稳定性特别强的因果链，就是“知识”。从智者那里学习“知识”，也有一个风险，就是他们有些人是从道听途说的经验中提纯知识，所以看似逻辑强大的知识，其实是错误的。最后，我们来讲讲创新。创新的本质，是一套“实践反馈系统”。毛主席说，实践是检验整理的唯一标准。我不知道什么是对的，那怎么办？那就试试。摔跤了，就是错了。找到果子了，就是对了。这就是实践反馈系统。实践反馈系统的好处是，直接。但是问题是，成本非常高。不可能所有的知识，都通过实践获得。否则，我们就在不断“重新发明轮子”。我讲了几件事：经验、学习、创新。经验，是历史反馈系统；学习，是他人反馈系统；创新，是实践反馈系统。哪样对你最重要？是的。他们都很重要。2019-07-29点赞数:62评论区：赵仰特 : 像高手过招，方可获得更多反馈，高手的种类和招数的种类越多，得到反馈后的我们就进步越快。过招反馈加工实践后，继续过招，如此往复，进步速度神速。可见反馈和很重要，谁给的反馈更重要。 有脾气 : 在信息极易获得的环境下，过滤三个反馈系统信息的能力应该是核心，否则很难解决未来的问题。 Howie : 没次出发都是 “回归均值” 周周 : 经验萃取和因倒技术 小张 : Pdca循环 好的保留，不好的删除 鱼本渔 : 历史经验的反馈系统最优化的建立方式=学习智者的反馈系统+实践反馈系统 Leona : 做事需要这三种反馈同时交叉进行，缺一不可。 柯索铝质家居 : 经验，是历史反馈系统； 学习，是他人反馈系统； 创新，是实践反馈系统。

从哪吒魔童转世

2019-07-28 #刘润
越檀—优势辅导教练 : 从《哪吒—魔童转世》里面看懂了什么？对于这部电影，启发了我对于独立人格意识，以及讲故事的领导力的感悟。1.培养独立人格意识，帮助孩子了解自己电影中敖丙的角色，大多数人认为他从小身负太多东西，从诞生之初就成了一场权利交易。（申公豹借他上位飞升金仙之位，龙族借他脱离东海龙宫牢狱）影片中：敖丙从小被各种意识绑架，开场不久就被老师追问今天修行的结果怎么样？现实中：父母亲戚对孩子都会问“最近学了什么呀，考试多少名？”影片中：敖丙缺少独立思考意识，师傅说需要为龙族未来而毁掉陈塘关全城，他认为合情合理。现实中：家里人给孩子灌输理念，你做什么，学什么才能过得更好。孩子也许被带偏觉得值得一试。影片中：哪吒PK敖丙时，对敖丙大喝“你是灵珠，怂什么，我命由我不由天”，强烈的独立意识觉醒。现实中：孩子在遭遇碰壁或者挫折时，也许就陷入了“我到底适合做什么”的哲学命题。影片中：敖丙最后放手一搏，与哪吒一起对抗天雷劫，甚至不惜动用家族授予的宝物，从而收获全新的自己。现实中：孩子在经受打击与困难后，在没有完全强大的独立人格意识前，往往选择了逃避和凑合着过。假若我是哪吒的话，我会问敖丙“这样的安排你过得开心吗？让你最有成就感的是什么事？”也许就这么一句，就帮助现实中的孩子打开了话匣子，给你一一道来他的所见所闻，以及他看待这件事的思考观点。那么如何培养独立人格意识？我认为有两步：A.聆听他们的内心声音；B.指导他们批判性思维的方法。聆听的时候，请注意千万别看手机，展示出你对这场对话的足够尊重与关注，孩子也会保持一个尊重你的态度，告诉你想知道的事情。指导批判性思维的方法，则比较繁琐，后面会再慢慢分享。建议阅读《批判性思维》进行实操，否则在面对未来复杂的生活，太多的信息量冲击着大脑，一觉醒来就处于价值观被颠覆的动荡之中。2.会“讲故事”的爸爸，能更有效影响自己的子女影片中：李靖面对哪吒的自暴自弃，给他编织了一个美好的故事，他告诉哪吒“你是灵珠（事实是骗哪吒）转世，出生之时天生神力，让村民心生忌惮，从而认为你是妖怪化身。你看我和你娘都是凡人，怎么会生出妖怪呢？我们希望你拜师学艺，就是为了帮助我们给村民去降魔伏妖，给这一方土地带来平安呢！”在这段情节里面，其实可以用“讲故事”的要素拆分。聆听者的渴望：哪吒希望被村民认可，注意，则经常捣乱搞得民不聊生。确认聆听者内心的渴望：李靖在看到哪吒对学法术眼里放光时，可确定前者的诉求。有清晰的障碍：来自村民对哪吒的偏见，那么做成这件事能够有效消除偏见。采取的行动：找太乙真人拜师学法术，这样能修心养性，既控制自己的戾气，把自己的潜力用正确的位置上展示情感：李氏夫妇对哪吒能认可这件事表现非常支持与鼓励，以增强哪吒的自信心。那么结果可想而知，哪吒听完都两眼发光，一转以往的顽固不化，乖戾成性的态度。可见做一个会“讲故事”的爸爸，其实就是在通过描绘蓝图以及提供解决方案来进行激励子女树立目标。现实中，我们同样把这个讲故事的套路用在与子女沟通上。例如遇到两者观点不合时，可用讲故事的方式来让气氛缓和且易于沟通，人天生就爱听故事，也能从故事汲取能量，而这些就是我们讲述故事的人，最希望的局面。2019-07-28点赞数:29评论区：深蓝_成&amp;渝 : 周五看完电影后感受颇深的是独立人格的培养的重要性。润总提到的“讲故事”的重要性，作为一个1岁多儿子的爸爸，看来需要继续修炼此法，和儿子共同成长。 一个老爸的自我修养 谢谢一路有润总和岛友们陪伴[胜利]
