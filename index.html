刘润 : 外界常把诺基亚对竞争激烈态势的无动于衷，解读为大公司高管的傲慢心态使然。诺基亚的失败，仅仅是因为高管强势，听不到来自下属的声音吗？这句话隐含的含义或许是：如果高管听到了下属的声音，这件事就会成功。但，真的是这样吗？ 在一些大公司，通常有自己专门的研究机构，比如“集团战略部”，“企业发展办公室”等。这些组织部门对未来趋势，市场反应，都有非常大量的数据和研究报告。同时还有一些赫赫有名的咨询公司守卫四方，共同来优化公司的战略发展能力和增长潜力。所以，诺基亚的高管，并不一定是听不到“外界的声音”。高管，其实能听到很多声音，也知道问题的根源。关键是，他们没有勇气颠覆自己。一个企业越成功，就意味着它的生态系统越完善，当它试图建立另一种生态系统的时候，就会面临极大的挑战，需要自己否定自己，颠覆自己，自己杀死自己。这，很难。所以，往往一个行业的变革者都来自行业外部。也许，最令这位昔日霸主沮丧的，不是失败本身，而是面对这早已预见到的失败，它的无能为力。为什么说，无能为力呢？外人站在外围去去看决策时，通常会觉得，这个决策不难做啊。可是站在CEO的角度，每一项决策都关系着各方力量的利益平衡博弈，既要意识到这个决策的好处，同时也要清晰的了解可能带来的风险和坏处。对于当时面临转折点的诺基亚来说，显然无法下决心放弃过去使公司大获成功的投资和资源。也更加留恋公司在过去产业秩序中的地位，无法放弃功能手机的优势市场地位，无法放弃当时占据着绝大部分市场份额的塞班系统。即：无法选择，从0开始。所以，诺基亚遇到的真正问题，也许并不是一个CEO听不到下属的声音，而是什么都能看得见，什么都能听得到，但是他却无能为力。面对不确定性，CEO无法掌握自己的“概率权”。那，什么人会掌握自己的“概率权”？只有身兼勇气，并且有权利，能够舍弃过去所有的人，才有可能。面对着市场大环境的极度不确定，在竞争高度激烈的全球市场中，每一项决策的延迟都可能是灾难性的。而一个职业经理人，通常是有这个勇气，但是没有权力代表股东舍弃过去的既有利益链条，而高效率旳成本控制思维，也没留余地让自己冒险。处于战略转型期的企业，一般选择将事关重大的部分业务独立出来，一方面体现其重要性，另一方面也可以防止旧业务的掣肘。但诺基亚却在早期“战略转折点”接连犯错，不仅没有把智能手机业务独立出来，反而将功能手机和智能手机业务合并，导致当时盈利占优势地位的功能手机部门统治了诺基亚的手机业务。大企业在试图进行内部创新时，往往“阻力重重&quot;，只有少数的企业才能够翻盘成功。这就是，决策博弈。通常不是在一个确定性和另外一个确定性之间做选择。这种选择，只要掌握的信息量足够，其实每个人都可以正确决策。决策之所以艰难，往往是在一个确定性和一个不确定性之间的博弈权衡。那时，我们需要的不是信息。而是，勇气。　　2019-09-03点赞数:65评论区：谢鹏 : “马后炮”！有什么用？ 楚楚 : 不仅仅是勇气，更多的是“结构” Howie : “生儿育女” 张晨 : 有随时重新开始的能力，而不是做好的决策的能力 slj : 利益平衡很难！打破既得利益集团，更难！ 郭惠 : 新的时代到来时，都会有新的机会产生，并不一定是市场领导者看不到变化与变革，很多时候是负担太重、掉头短期代价太大很多时候甚至需要去颠覆自己。新入局者0负担，就容易出现颠覆。 刘萍 : 个人面对改变的恐惧如此，何况于一个巨大的公司。 A佳源达-kevin : 对新生事物失去的警惕性，消费习惯的改变和臃肿的机构决策

2019-09-03 #生财有术
阿狸 : 又有一段时间没上星球了，还是在做之前上一篇帖子中说到的那个开淘宝店卖头像印章产品，9月1号全国幼儿园开学，所以上周很忙，虽然并没有赚很多，至少星球的会员费是赚了几倍，也算是一次工作外赚钱小道的初步尝试吧。 先放些店铺数据上来，再接着说。（图1） 从6月底开店到现在2个多月，主要做了哪些事呢？ 1.找到了一个很细分的产品，因为本身比较喜欢画画，所以这个产品算是淘宝上独家，又恰好赶上幼儿园开学这个时机，让我觉得做淘宝似乎并不难，但是要做到品质稳定、服务稳定最后收获稳定的销量却不容易； 2.2个月期间，不断测了大概20多个主图（图2），排名决定曝光，主图决定访客数； 3.不能避免肯定要刷数据，开店第1个月只找朋友帮忙sd，自然流量很少，第2个月开始人为干预，先是干预了2周的访客数，做的很保守，第3周开始干预转化率，就是专业刷手。因为产品本身需要人工绘制头像，所以1天也就刷几单，不然忙不过来，白天上班下班带娃，只有深夜画图； 从结果上来看，sd确实很重要，特别在前期流量很少的情况下，数据基础太低也对主图优化、详情页优化起不到分析的作用，通过sd后进来的自然流量以及和客户交流，才会知道客户在意的点在哪里，是不是我的目标人群（主要是价格方面）； 另外，一件代发最重要的是能做到货源品质稳定、客服及时回复，我觉得就可以了，在找货源时筛选老店一般都不会有什么问题，不然售后会很麻烦，一个差评就毁了所有。 幼儿园开学后，订单明显减少，家长们都没啥需求了，这件赚钱的小插曲也算告一段落，后面可能会继续寻找别的产品，或者从画画上再发展出其他变现可能（并没有经过专业美术学习，纯兴趣爱好）。 总结一下吧，还是那句话，执行力比再完美的方案都重要，只要有可行的想法，在做的时候才会知道要遇到哪些坑，要怎么去解决，那些棘手的难题，无从下手的乱麻，最后都会慢慢推进，水到渠成。感谢生财有术，感谢亦仁，感谢不记得是哪个星友偶然的一条关于印章的评论让我迈出了工作之外赚钱的第一步。下一件事，把这段时间错过的精华帖补完。2019-09-03点赞数:39评论区：亦仁 : 抖音印章案例，那位分享的圈友是我[胜利] 阿狸 : 😂 😂 😂 感谢亦仁大大 伊迭 : 看再多精华不如做一个案例 吴亚军 : 重在执行 1只V : 这个产品可以做出来更大的量，本地就有一个，赚的手软 阿狸 : 加我微信：chengzides，聊聊
