Long Tail Pro早期是请外包开发的，也掉进了一些坑，不过后面跟优秀程序员沟通时，也学习到了一些经验。 首先，在雇用程序员之前，尽可能详细沟通。 并尽可能记录对每一个任务的期望；其次，询问潜在的雇用对象对目前的项目的问题或者看法，如果他们没有任何问题，可能是没经过思考，从雇用的成功经验来看，优秀的程序员总是能够提出更深层次的问题或者提出作者没有想到过的问题。第三，讨论对于后续维护和修复bug的期望，每个产品都会出现错误，因为需要了解程序员他们是否愿意持续维护这些产品，以及对于补偿的预期是什么，比如Long Tail Pro就是一次性支付了固定的工作价格，但是后续的维护则按照小时支付给程序员工资；第四，一般来说更贵的程序员做的更好些，作者雇用过的最贵的程序员，在过去几年也确实是表现最好的那位，每一分钱都非常值；二. 同时用慢速和快速的办法来推广产品慢方法：刚开始推出LongTail Pro的时候销量很暗淡。 刚开始给1000个邮件订阅者发放了邮件，并在Warrior Forum发了Long Tail Pro的优惠码。软件早期的买家也是他博客的读者，作者这边特别强调了一个主要的营销策略：写真正有价值的干货，找到忠诚的追随者，然后用软件去解决他们的问题。 （想到了谁？顾小北老师有没有!）虽然博客或者写公众号是一种较慢的办法， 在Long Tail Pro成为真正的生意之前花了几年时间， 但是目前作者的博客仍然是软件销量的第一渠道，甚至超过了所有的联盟营销公司。快方法：Long Tail Pro经过几年的更新，在2012年的时候，差不多每个月可以赚10000美元。 这时候作者准备再正式地发布产品，搞大事情！第一件事：找到一位联盟经理，让他帮助招募联盟营销公司并给出特别优惠，看看会发生什么，销售额是不是会提升。 当然，成功爆掉了！重新发布多次超出作者的预期，还成功打破了JVzoo和论坛特别优惠的各种记录。 2019-08-15 #刘润
海东 : 受到润总的“红利、社会工资、创新利润”（原文链接：）启发，解答了一些商家的困惑，大家反馈还不错，贴出来给大家看看。（承蒙润总照顾，已被设为精华[抱拳]）[得意][得意][得意][得意][得意][得意][得意][得意][得意][得意][得意][得意][得意][得意][得意][得意]《强力对手打压，我该如何破局？》看到这个探讨，我也感觉很闹心，大家说的那些操作手法，我都不擅长，我打算从另一个角度，说说这个事。不知大家知不知道，这个世界上，你赚的钱分三种：1、红利；2、社会工资；3、创新利润；我们赚的钱，逃不出这3种，只是我们可能没有好好去分析，我们到底属于哪一种。我相信，我这个分析，对今天这个求助者是有帮助的，从宏观的层面，让当事人明白，我们的产生困境的宏观原因。第一种：你赚的是红利的钱；什么是赚红利的钱？就是你这个市场，刚开始是短暂的供需失衡，即竞争者太少，需求相对大，然后你赚了这个钱，赚得很爽，很多创业者是处于这个阶段，而不自知；你赚得了这个钱，不是你太牛叉，而是这个市场供需失衡，你抢得了头汤；毕竟钱赚得很爽着呢，赶紧抢钱要紧，哪里听得了我这类的废话？哈哈。请大家对号入座一下，你现在是不是处于在赚红利的钱，赚的爽的同时，要居安思危哟！第二种：你赚的是社会工资；在这种阶段，创业者其实就是给社会在打工，你也是社会大分工下的一个岗位而已，是社会在给你发工资；这个阶段是怎么产生的？就是因为，你赚得挺爽的，其它竞争者也想进来抢一点爽一会，然后竞争者越来越多，利润摊薄到你手里，就是少得可怜，然后你就沦落为给社会打工的，领取微博的薪水；为什么竞争者一冲进来，就是这个样子了呢？就是因为你在吃红利的时候，并不能认知到自己在吃红利，自己并没有什么核心竞争力；当门口的野蛮人闯进来了，你可能还不能明白，到底发生了啥？我怎么就赚不着这个钱了，我明明很努力啊。。。比如说，当事人做的这个插座，对手总是比你低2元，抄你的图，你的详情，模仿你的产品，并进行了压缩成本的操作（把线做得更细），然后，就把你的顾客给抢走了。其实你这个对手的日子，也并不会好到哪里去，大家都是争抢这最后的利润，赚一波再说；那么，在这个领取社会工资的阶段，你是无法赚取到真正的利润的；因为无非是这个市场上多了很多竞争者，而顾客还是那么多，不管你如何折腾，最终你的利润，就是大家一起摊，到你手里的都是在减少；遇到那种战略亏损的对手，你更没有机会。那么，怎么解决？一定不是刷刷单，抄抄图，就能够彻底解决的，我们必须要想办法跳出第二个领取社会工资的阶段，才行。我换个方式说，如果今天没有这个对手跟你恶搞，大家都是卖一样的价格，你同样面临利润微薄的局面。因为更多商家会冲进这个市场，都想分一杯羹，大家都守规矩，然后市场这个大老板，给大家分钱，到手的都会下降；反过来说，今天你是那个对手，同样的手法搞别人，也并没有日子好过哪里去，他迟到也要面临你这个困局。我们这时候，应该想的事，如何换一个赛道玩，否则痛苦是避免不了的。第三种：真正的利润来源（创新）；这里的创新，不仅仅是产品的创新，还包括运营能力的创新。1、 你能否在品质不变的情况下，让你的运营成本更低？其实，你为什么跟不了价，就是因为你你的运营成本还是很高，比如人员工资、营销成本、办公室成本、供应链成本、等等各种成本，你如果有办法做到优化，一定是可以把成本降下来的，这个时候，你成本降下来的，用同样的品质，再去对付这个降质又降价的对手，我相信是有打赢的胜算的；2、 产品创新，而且让对手很长一段时间无法跟上。比如，有专利，比如有技术难度；也是可以解决这个苦恼；商场的竞争，就是如此，都是需要有硬实力的，不是搞搞什么运营技术，就能解决的，毕竟你最终要靠你的商品来比拼。3、 创新后，产生的利润，才是真正的利润；如果你搞不定，红利也吃不着了，然后被迫处于领取社会工资这个阶段，那痛苦是会一直跟随着的。理想的状态是：我们创新，开始吃红利；持续创新，继续吃红利！以上，就是我的观点，可能扯得比较大，大家并不一定受用；可能大家会说，我现在就是没有能力进入第三阶段，你说怎么办吧？我肯定没有办法，我相信你也没有办法，用这种抄袭恶搞的办法，是没有办法打跑更多对手的；只有提升运营效率（降低成本）和产品创新（提升溢价），有了更大的突破，才能解决问题；我们无法摆脱商业市场的规律做生意。[微笑]今天的观点，大家有一点启发，我也欣慰啦~权当抛砖引玉吧，其它人继续！https://t.zsxq.com/vZvFY37 刘润5分钟育儿

2019-08-11 #刘润
刘润 : 《刘润•5分钟育儿学堂》我和小米一起做饭，突然我的电话响了。一个陌生号码。我的手是湿的，所以用指关节按下接听和免提。电话那头传来一个训练有素的声音：&quot;您好，我是银行贷款中心的，请问……&quot;我用指节挂了电话。小米有点吃惊。我说这是个推销电话。小米问，为什么呢？万一是有事找你的呢？我觉得有必要给小米讲讲&quot;同理心&quot;，于是说：小米，你很喜欢看名侦探柯南。但是你知道，一个侦探最重要的能力是什么吗？小米问：是什么？是同理心。同理心，是妙手神探，和他的死对头，连环杀手，共通的底层能力。什么是同理心？同理心，英文叫empathy，是知道别人在想什么的能力。同理心，和同情心不同。同情心，是共享别人情绪的能力。同理心，是看透别人的理性（动机，逻辑，计划）；同情心，是共享别人的感性（快乐，痛苦，忧伤）。我问小米，比如刚才那个电话。如果确实是我的借记卡、或者信用卡所在的银行，打电话有事找我，你觉得他们会怎么说？小米想了想，说：你好，我是……是。我说：我是什么？小米说：我是招商银行信用卡中心，请问你是刘润先生吗？是的。我说。你会说：&quot;我是银行，请问……&quot;吗？小米想了想说，不会。这就像你打电话给你的朋友，你会说：我是小米，请问XXX在吗？你做自我介绍时，你会很自然说清楚自己是谁。招商银行打电话来，会说我是招商银行，不会说自己是银行；中国移动打电话来，会说自己是中国移动，不会说自己是手机公司；海尔公司打电话来，会说自己是海尔，不会说自己是家电工厂。因为，招商银行，中国移动，海尔公司，才是可以用来表明身份的自我介绍。这是正常的逻辑。一个电话打来说，&quot;我是银行贷款中心&quot;的。这个介绍，没有表明身份，却想从你那里获得信任感。说明她很可能并不属于任何一家银行。她属于任何一家银行，都会说&quot;我是XX银行&quot;，而不是&quot;我是银行&quot;。那她是什么？她可能是推销，甚至是骗子。所以我挂了电话。站在对方的动机、逻辑、计划的角度思考，这就是同理心。小米说：哇，同理心这么厉害。我对小米说，挂掉推销电话，可能只是同理心的最微不足道的价值。妙手侦探要靠同理心破案。一个人推门进来，他的动机、逻辑、计划是什么？然后呢？再然后呢？连环杀手要靠同理心作案。一个人推门进来，他的动机、逻辑、计划是什么？我这样，我那样…同理心，是一种非常强大的能力，它是很多能力的元能力。演讲能力，沟通能力，销售能力，管理能力，辩论能力，谈判能力，等等，都是从同理心上长出来的二级能力。有同理心的人，可以通察人心。慢慢练习吧，小子。2019-08-11点赞数:110评论区：Emily.Z🍭 : 小米从小训练有素，长大不得了… 菊姐 : 这样的家教，没有学校教的了这样的孩子 郭惠 : 从小每天接受中国最贵的商业顾问之一的辅导，是怎样一种体验[偷笑] 王清兮 : 记得儿子三岁的时候，我没有管他懂不懂就是单纯跟我学：“像不等于是！”“别人做错不是我做不对的理由”。小家伙一开始只是大舌头的发音学，然后我就讲给他听。有一天他突然说，妈妈你看这里很像我的幼儿园，但不是。今年孩子9岁了，他在对事物逻辑分析上不是同龄孩子的思维了。有自己的认知和看法。我给他讲我的工作从来都是像和大人讲话那种，6岁那年我和他讲我的促销模式，小家伙一本正经的说，“妈妈，你让别人挣到钱你才能挣到钱。”[呲牙]教孩子模仿，不如教孩子思考！刘润老师在教孩子“小事儿背后大逻辑”这值得我们做父母的学习！今天我也学到了同理心和同情心的巨大差别，受益！ 王清兮 : 学校只是基础知识教育，不能期待很多。真正差异化的教育来自家庭。 深蓝_成&amp;渝 : [强] 小张 : 把自己放进别人的鞋子里，用对方观点说服自己 先归纳要点，再用演绎法演绎对方的观点。 张小胖 : 这也算是一种筛选顾客，能接着往下聊的才可能是目标对象。
